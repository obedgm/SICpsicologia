Logo and Navbar

To change the logo, replace the "logo.png" file in the assets/images folder. The default logo size is 184x28 px. As you scroll down the page, the navbar shrinks and the logo becomes smaller.

.navbar-brand img { width: 184px; }

Navbar Default

Navbars are responsive meta components that serve as navigation headers for your application or site. They begin collapsed (and are toggleable) in mobile views and become horizontal as the available viewport width increases.

Change the text and links of navbar:

<li><a href="index.html">Home</a></li>

Creating Dropdown Menus

The dropdown menu is typically used inside the navigation header to display a list of related links when a user mouse hover or click on the trigger element.

You can use the Bootstrap dropdown component to add toggleable dropdown menus (i.e. open and close on click) to almost anything such as buttons, navbar, tabs and pills nav etc. without even writing a single line of JavaScript.

The following example will show you how to add dropdowns to navbar.

<li class="dropdown">
<a href="#" class="dropdown-toggle" data-toggle="dropdown">Blog <span class="caret"></span></a>
<ul class="dropdown-menu wow fadeIn" data-wow-duration="500ms" role="menu">
    <li><a href="blog.html">Blog Sidebar</a></li>
    <li><a href="blogpost.html">Blog Post</a></li>
    <li><a href="error404.html">404 Error</a></li>
</ul>
</li>

Home Page

Home Slider

Change the pictures and the text in the slider of the home page.

Change all pictures in the stylesheet file (main.css) or in the SASS file (_images.scss)

.bg-slider-wrapper .flexslider .slide.slide-1{
    background:#033f58 url('http://placehold.it/1920x600/BDC3C7/ffffff') no-repeat 50% top;
    -webkit-background-size:cover;
    -o-background-size:cover;
    background-size:cover;
}

Change the text of slider in the home page

<section id="home-promo" class="home-promo">
    <div class="container text-center wow fadeInUp" data-wow-duration="1000ms" data-wow-delay="500ms">
        <h2 class="titlepro">
            <span class="upper">Welcome to</span>
            <span class="middle">Corpboot <strong>Template</strong></span>
            <span class="bottom">A Bootstrap website template for business</span>
        </h2>
        <button class="btn promo-button" type="button" data-toggle="modal" data-target="#modal-contact" data-backdrop="static"><i class="fa fa-download"></i> Download</button>
    </div>
</section>

Note: If you do not want to use html text in the slider you just have to comment out or remove this snippet.

Change settings in javascript file (main.js - line 6)

if($('#bg-slider').length) {
    $('#bg-slider').flexslider({
        animation: "fade",
        slideshow: true,
        animationLoop: true,
        directionNav: false, //remove the default direction-nav
        controlNav: false, //remove the default control-nav
        slideshowSpeed: 6000,
    });
};


More properties: https://github.com/woothemes/FlexSlider/wiki/FlexSlider-Properties

Google Maps and Contact Form

var myLatLng = {lat: 40.7060555, lng: -74.0090263};

Use the class "selectpicker" and the data-style "selectcorp" if you want to create more fields of this type, to maintain the same style of form design. Example:

<select class="selectpicker" data-style="selectcorp">
    <option value="Option 1">Option 1</option>
    <option value="Option 2">Option 2</option>
</select>

$youremail = 'youremail@domain.com';

Names of the form fields

In the Contact Form:

  • departament (select)
  • subject (input-text)
  • name (input-text)
  • email (input-text)
  • message (textarea)

In Blog Post, for the comments form:

  • name (input-text)
  • email (input-text)
  • website (input-text)
  • message (textarea)

To add a new validated field, just copy and paste the following code, replacing the "for" values in the label tag and the "name", "id" and "data-name" values in the input tag.

<div class="form-group">
    <label class="sr-only" for="phone">Subjet<br></label>
    <input type="text" name="phone" id="phone" value="" class="form-control required" placeholder="Phone" data-name="Phone">
</div>


If the field is not required delete the "required" class.
